# Connect social media

Connect is a facebook inspired social media platform created with React/Redux and Hooks!

# Stack:
Front-end:<br />
React (Hooks), Redux, React router, React Webcam<br />

Back-end:<br />
Nodejs, Expressjs, Express Router, Socket.io<br />

Database:<br />
Postgres SQL<br />
